# Crypto Glossary 📚

A curated collection of crypto, DeFi, and blockchain terms with definitions and examples.

## Examples
- **Staking:** Locking crypto in a protocol to earn rewards.
- **Liquidity Pool (LP):** A smart contract pool used for decentralized trading.
- **APR/APY:** Annualized yield metrics.

## Goals
- Help newcomers understand crypto concepts
- Add examples for clarity
- Update regularly as new terms emerge
